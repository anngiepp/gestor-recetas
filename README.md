# 🎬 CineGestor — Gestor de Películas

## 📖 Descripción
Aplicación web desarrollada en Django que permite administrar una colección de películas y sus directores. Permite crear, editar, eliminar y buscar tanto películas como directores, con una relación entre ambas entidades.

## 🛠️ Tecnologías utilizadas
- Python 3.x
- Django
- SQLite3
- Bootstrap 5
- Bootstrap Icons
- HTML5 / CSS3

## ⚙️ Pasos para instalación y ejecución

### 1. Clonar el repositorio
```bash
git clone https://github.com/anngiepp/gestor-peliculas.git
cd gestor-peliculas
```

### 2. Crear y activar el entorno virtual
```bash
python -m venv env
env\Scripts\activate
```

### 3. Instalar dependencias
```bash
pip install django
```

### 4. Aplicar migraciones
```bash
python manage.py migrate
```

### 5. Ejecutar el servidor
```bash
python manage.py runserver
```
